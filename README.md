## 介紹

這是一個可以拿來練習前端 CRUD 的後端專案，可在本地使用，也可自行部署到雲端服務。

> origin 的限定可自行調整 src/main.ts 相關程式碼。

## 下載與啟動專案

```bash
$ git clone https://github.com/JiaHongL/todomvc-backend.git
$ cd jwt-refresh-token-mock-backend
$ npm install
$ npm run start:dev
```
